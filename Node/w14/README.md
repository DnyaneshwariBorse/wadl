# Node.js W14 Project

A simple REST API and frontend setup.

## Dependencies
- Node.js
- Express
- CORS

## How to Install
To install dependencies for this specific project or general Node projects:
```bash
cd Node
npm install express cors
```

## How to Run
Run the backend using node:
```bash
node w14/backend/server.js
```
The server will start listening on the port configured in `backend/server.js`.
