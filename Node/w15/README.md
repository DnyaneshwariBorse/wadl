# Node.js W15 Project

An Express-based API server with a static asset frontend.

## Dependencies
- Node.js
- Express
- CORS

## How to Install
Install the dependencies using:
```bash
cd Node/w15
npm install express cors
```
OR
```bash
npm install
```

## How to Run
Start the API backend using:
```bash
node backend/server.js
```
Open `frontend/index.html` via Live Server or a static file server to test the frontend.
