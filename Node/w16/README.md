# Node.js W16 Project

An Express-based API server with static web page routing.

## Dependencies
- Node.js
- Express

## How to Install
Install the dependencies using:
```bash
cd Node/w16
npm install express
```
OR
```bash
npm install
```

## How to Run
Start the backend server using:
```bash
npm start
```
OR
```bash
node backend/server.js
```
The server will automatically serve the files inside the frontend directory.
