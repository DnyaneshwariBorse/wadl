# Bookstore App

A complete Node.js/Express and MongoDB bookstore application.

## Dependencies
- Node.js
- Express
- Mongoose
- CORS
- A local or remote instance of MongoDB

## How to Install
Run the following command to install dependencies:
```bash
npm install express mongoose cors
```
OR
```bash
npm install
```

## How to Run
1. Ensure your MongoDB service is running.
2. Start the backend server:
```bash
npm start
```
The application runs on the port specified in `backend/server.js` (usually `http://localhost:5000` or `http://localhost:3000`).
