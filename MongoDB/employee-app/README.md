# Employee App

A CRUD application built with Node.js/Express and MongoDB.

## Dependencies
- Node.js
- Express
- Mongoose
- CORS
- A local or remote instance of MongoDB

## How to Install
Run the following command to install dependencies:
```bash
npm install express mongoose cors
```
OR
```bash
npm install
```

## How to Run
1. Ensure your MongoDB service is running.
2. Start the backend server:
```bash
npm start
```
The application runs on the port specified in `backend/server.js`.
