PICT B.E Course Structure

First Year (FE)

Mathematics I: Basics of calculus

Physics: Mechanics and waves

Programming in C: Fundamentals of coding

Engineering Graphics: Drawing concepts

Second Year (SE)

Data Structures: Arrays, linked lists, trees

Discrete Mathematics: Logic and graphs

Computer Organization: CPU and memory

OOP: Object-oriented programming

Third Year (TE)

DBMS: Database management systems

Operating Systems: Process and memory

Computer Networks: Networking concepts

Web Development: Frontend + backend

Final Year (BE)

Machine Learning: AI basics

Cloud Computing: AWS concepts

Big Data: Data processing

Project: Final year project
